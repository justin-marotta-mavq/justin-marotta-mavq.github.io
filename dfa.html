Dashboard Download Options

PDF: Save one or more dashboards or workbooks at a time as a PDF (multiple layout options available).
Crosstab: Download CSV file (each chart or table will need to be chosen and each data set exported separately).
Data: Open a web page with the visualizations data in a table and download as a text file from there. (Make sure to select the visualization for which you want data in the view. Full data tab within the download page will provide the best information.)
Image: Save the entire visualization as a JPEG file.
If the Data/Crosstab options are not available, click on one of the exhibits and then attempt the download.
